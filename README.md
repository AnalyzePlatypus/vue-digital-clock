# vue-digital-clock

> A digital clock component built with Vue.js

## Example

```vue
<template>
  <Clock />
</template>

<script>
import Clock from 'vue-digital-clock'

export default {
  name: 'app',
  components: {
    Clock,
  },
};
</script>
```
